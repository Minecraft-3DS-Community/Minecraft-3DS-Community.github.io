About the Unistore
The Minecraft 3DS Community Unistore contains a catalog of community-made mods, worlds, and other content - avaliable for download directly from your console via Universal-Updater.

GitHub Repository
Universal-Updater

Installation
  1. Open Universal-Updater
  2. Navigate to Settings → Select Unistore → Add
  3. Open the Recommended Unistores page
  4. Select the Minecraft 3DS Community Unistore.

For a more extensive guide, see the guide on the wiki or on the Community Discord.
