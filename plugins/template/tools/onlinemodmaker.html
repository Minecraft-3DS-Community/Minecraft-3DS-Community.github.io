This online tool is a mob/player behaviors editor. It includes all mobs from the latest version of Minecraft: New Nintendo 3DS Edition. You can edit these JSON files and apply them to your mob of choice by placing them in the following folder:

\luma\titles\[TitleID]\romfs\resourcepacks\vanilla\server\entities

If you do not know your TitleID, you can find it listed below.

USA - 00040000001B8700

EUR - 000400000017CA00

JPN - 000400000017FD00
