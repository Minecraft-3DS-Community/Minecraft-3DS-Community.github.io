How to Dump and Extract RomFS

Prerequisites

Your console needs to have custom firmware (CFW) installed. If you do not have custom firmware installed, please go to https://3ds.hacks.guide and follow the guide for installing it there.

Make sure the latest version of GodMode9 is installed. It can be downloaded here.

To open it, make sure the console is powered off, then press the Start button to load the Luma chainloader menu. Select GodMode9 to launch it.

If you want to dump your cartridge, make sure it is inside of the system.

Dumping

Dumping the base game (version 1.0) is not necessary but should be done just in case. Its RomFS is not referenced much when modding, but some mods, like importing custom music, require dumping it.

You will need to dump the game's update (1.9) to for most mods as the final game mostly references the update's file structure.

Dumping the Cartridge (1.0)

  1. Open GodMode9
  2. Navigate to [C:] GAMECART
  3. Press A on [TitleID].trim.3ds
  4. Select “Copy to 0:/gm9/out”

Dumping the Base Game (1.0)

  1. Open GodMode9
  2. Navigate to [A:] SYSNAND SD
  3. Select "title"
  4. Press the Right Shoulder button and A
  5. Select "Search for titles"
  6. Select [TitleID] Minecraft: New Nintendo 3DS Edition (KTR-BD3E-

Dumping the Update (1.9)

Follow the same steps as for the base game but locate the update title instead.
